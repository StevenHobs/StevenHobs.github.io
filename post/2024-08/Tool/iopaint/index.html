IOPaint 人工智能修图整合
2024-08-04 22:48:22 # 工具软件 #人工智能 #设计 #电商

简单介绍

iopaint 修图工具，这个软件最直接的用法就是去污点，抹除细小的东西。支持很多插件，有老照片超分辨率修复，遮罩图生成，以及Stable Diffusion AI的物体生成。

整合的思路

  1. 使用pyenv提取出来的python运行时，python 3.11可行；

  2. 先安装iopaint插件所需的依赖库，再安装 torch及iopiant；

  3. 使用git仓库克隆的iopaint代码时，要将web_app的前端代码编译至assets下；

  4. 使用环境变量 XDG_CACHE_HOME 来指定模型缓存路径；

  5. 通过bat脚本执行iopaint配置命令和iopaint工作台命令；

使用

下载对应的整合包，安装支持7z的解压工具，这里推荐使用 Nanazip 微软应用商店可装，

解压的时候注意路径位置不要包含压缩包的名字，不需要，路径直接就是 D:\

随记

做整合包真的很耗时间，模型得一个个下载好，整个压缩包都好几个G大小。然而自己却不怎么使用，希望之后能销售好点，也不枉我折腾了半天时间。
